其他问题

1.npm install安装依赖报错，无法安装：

可以使用npm需要访问国外网站比较慢，可以使用国内镜像提升下载速度，命令行执行如下：

 npm config set registry https://registry.npm.taobao.org
 //后面执行cnpm install
 cnpm install

2.gulp dev报错throw error：

查看代码语法是否有误，需修改代码后重新命令行执行gulp dev
